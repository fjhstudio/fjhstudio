## Activities

fourjoehalf Studio（四畳半スタジオ）は、Arata Ouchi（オオウチアラタ）の個人制作プロジェクトです。私の小さな部屋から、さまざまなプロジェクトが生まれ、さまざまなプロダクトが生まれ、今では多くのコミュニティを支えるデジタル・プロダクトやSaaSを開発しています。個人事業主としての活動は[Original SIN Architecture](https://osa.xyz)
で行っています。

## Contribution
